Unity-chan: Endless Runner

  • September 9th, 2016
  • Complete

Project Info

This is simple endless runner game starring the free asset character Unity-chan. It features procedural generation of background and obstacles and saves the highest score into a file using serialization. This project allowed me to practice the techniques freuently used in endless runner games.

The gameplay is very simple. The objective is simply to jump over the incoming obstacles as the character continues to run down the path. Pressing jump briefly will make Unity-chan jump at normal height while holding jump for more than 0.5 seconds will cause her to jump higher for avoiding taller obstacles. Upon death, the score will be saved as a high score if it was higher than the previous high score. This score will then be stored inside a file and loaded again on the next playthrough as the high score.

  • Learned procedural generation with obstacles
  • Good practice with endless runner style of game design
  • Learned to do save data management using serialization
  • Used limited assets to create a basic project

Skills:
