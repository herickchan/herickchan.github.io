All Hallow's Eve - Revenge

  • Janurary 23, 2011
  • Complete

Project Info

This is another old project using Java when I first started to make games. This game is a zombie survival game where players must shoot down on incoming waves of zombies from the top of a house (which acts as the players "HP"). In between each wave, players can upgrade their weapons or repair the house. The enemies increase in HP as players progress through the waves and take more damage if shot in the head compared to the body. This game has a very basic local multiplayer system where two players can cooperate to take down the zombie waves.

  • Learned basic local multiplayer coding and logic.
  • First time making a zombie shooter.
  • More practice with game design and editing art assets.
