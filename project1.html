Final Fantasy 7 - Mini RPG

  • October 4, 2010
  • Complete

Project Info

This project is where I first learned how much I loved making games. This game is a simple RPG inspired by Final Fantasy 7. The gameplay consists of a top down overworld where the player can encounter treasure, enemies and eventually, bosses. The battles are turn based and the player gains new skills as they progress through the game.

  • Learned basic game design. (I made the mistake of making this game too difficult)
  • Used image editing software to put together UI and other important game visuals.
  • First time making a game with Java. (Although using the outdated HSA library)

Skills:
